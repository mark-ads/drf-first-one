1) Установить  drf
 - установить django-debug-toolbar
 - установить drf-spectacular
 - установить drf-nested-routers
 - установить django-filter
 - установить django-stubs
 - установить markdown

2) Создать проект
 - django-admin startproject first_one . 
 - cd first_one

3) Создать приложение
 - django-admin startapp first_app
 - cd ..

4) Задать настройки DRF в settings.py
 - в самый низ настроек

	Настройки для работы DRF

	INSTALLED_APPS += [
		'debug_toolbar',
		'rest_framework',
		'first_one.first_app',
		'drf_spectacular',
		'django_filters',
	]

5) Создать url для авторизации
 - в файле urls.python, в список urlpatterns

    path('api-auth/', include('rest_framework.urls')),

6) Сделать модели
 - в файле models.py

7) Пока опционально. Сделать миграции
 - python manage.py makemigrations
 - python manage.py migrate

8) Сделать файл популяции БД
 - project/app/management/commands/__init__.py
 - project/app/management/commands/populate_db.py

8.1) Опционально Создать суперюзера
 - python manage.py createsuperuser --username admin --email admin@example.com

9) Провести популяцию БД
 - python manage.py populate_db

10) Создать сериализаторы
 - project/app/serializers.py

11) Создать права
 - project/app/permissions.py

12) Создать вьюшки
 - project/app/views.py

13) Добавить роутеры
 - project/urls.py
 - project/app/urls.py

14) Добавить фильтры
 - project/app/filters.py

15) Подключить фильтры к вьюшкам

16) Добавить celery
- project/celery.py
- project/app/tasks.py
